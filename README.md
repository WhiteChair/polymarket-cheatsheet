# Polymarket Cheat Sheet

Mobile-first cheat sheet & calculator for amateur Polymarket bettors.

## What's in it
- Price → payout multiple reference table
- Scalping gain tables (mid-range + longshots)
- Amateur traps card (Rule 1 + Rule 2 + liquidity deep-dive)
- Polymarket fee breakdown by category
- 10-second pre-bet checklist
- "Should I take this bet?" calculator with edge, EV, fees & shares-received

Single static HTML file, no build step, no tracking.

## Deploy
Static site. Vercel auto-detects. Just connect the repo.

## Not financial advice
Polymarket involves real risk. Rule 1: bet what you can afford to lose.
